# Ticketing System Lab
<h2>Description</h2>
This lab demonstrates the setup and use of a Jira Service Management ticketing system to simulate real world IT help desk operations. The project includes the creation and resolution of mock incidents to practice ticket intake, prioritization, troubleshooting workflows, and documentation.
<br />


<h2>Tools Used</h2>

- <b>[Jira Service Management](https://www.atlassian.com/software/jira/service-management)</b> 
- <b>ChatGPT</b>
